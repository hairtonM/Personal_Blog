¿Qué es CSS3?

En esta sección vamos a hablar de ¿Qué es CSS? y ¿Qué es en realidad CSS3?

CSS, es un lenguaje de estilos y diseño que permite a nuestra website y aplicaciones web, poder tener un diseño no solo con estilo y un buen concepto del color. Además permite hacer Responsive Web Design.

Ahora bien en CSS se trabajan por módulos en donde cada módulo tiene diferentes versiones. El nombre CSS3, es un nombre comercial ya que de manera veridica y pública no hay una versión 3 de este lenguaje de estilos.

Ahora bien cada módulo en CSS te permite tener mejores cosas y funcionalidades al momento de diseñar y maquetar una web, para mayor información visita el siguiente link Módulos CSS.
