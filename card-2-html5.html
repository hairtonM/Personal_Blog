HTML y Las API's

En esta sección analizaremos sobre cuales son las mejores que trae consigo HTML5.

Primero vamos a definir que es una API conocida tambièn como Application Programming Interface que es una interfaz que define la interacción entre diferentes intermediarios de software.

En el caso de la nueva mejora de HTML definida comercialmente como HTML5. Esta nueva versión de HTML5 trae una estructuración de nuestra website de una manera màs simple y unas API's de JavaScript para manejar y manipular Audio, Video, podemos dibujar con Canvas en 2D y 3D tendremos acceso al hardware y a sus datos localStoraged y mucho más esto lo verás en el siguiente link: https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5.

Por lo cual HTML5 son 2 cosas semántica y API´s.

En el caso de la semántica trae nuevos tags(etiquetas) para la estructuración header, nav, section, aside, footer y otras etiquetas para formularios form, además introduce Las matemáticas por ejemplo MathML, y las etiquetas de audio y video.

En conclusión la nueva versión de HTML5 trae consigo muchas nuevas caracteristicas y buenas oportunidades de hacer una website no solo muy vistosa sino con una estructuración bien definida y segmentada.
