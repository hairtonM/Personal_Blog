Estructura principal de HTML

En esta sección del curso de HTML5 hablaremos sobre la estructura principal de HTML, responderemos preguntas cómo ¿Qué son las etiquetas?, entre otros.

Primero debemos definir muy bien ¿Qué es HTML?, partiendo de diferentes definiciones por entidades reguladoras de HTML como W3C (World Wide Web Consortium y otras como la W3schools. Definimos HTML como "Un lenguaje de estructurado, marcado y definición de la información de páginas web"

Ya definido HTML, empezemos por las etiquetas o también conocidas como tags, todo en HTML, se trabajan con equitas, ahora bien hay etiquetas de apertura y cierre. A continución te mostraré como se estructura una etiqueta.

Cada elemento en html tiene una etiqueta de apertura y de cierre y entre estas dos va el contenido. En ejemplo es una etiqueta de parráfos "p".

Ahora bien existen los Atributos (clases, identificadores), las etiquetas pueden tener atributos que más adelante nos permitira darle estilos de acuerdo a las clases. Sin embargo, daremos una visión general en esta sección.

Las Clases, nos permiten otorgarle un atributo a una etiqueta en html para luego referenciar a ese elemento con esa clase para otorgarle algunos estilos con CSS (Lenguaje de Estilos en Cascada), o también se usan para darle algunos eventos o funcionalidades con JavaScript (Lenguaje de Programación Orientado a Objetos).

Los identificadores, por otra parte también se pueden usar para dar estilos, sin embargo son recomendados para declararse una sola vez en cada etiqueta, mientras que una clase pueden ser utilizado en más de un elemnto HTML. Ahora bien los identificadores estan definidos por el siguiente ejemplo:
