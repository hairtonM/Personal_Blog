CSS en HTML

En esta sección vamos a hablar sobre las formas que hay para usar CSS en la estructuración de nuestra website con HTML y analizaremos ¿Cuál es la mejor?.

La razón por la que usamos CSS en HTML, es para hacer nuestra website mucho más visible y con mayor diseño y presencia, en pocas palabras para que se vea lindo.

Primera forma "Usar CSS dentro de las etiquetas de HTML", esto era muy común en los inicios de la web sin embargo es poco ventajoso al momento de querer reutilizar algunos estilos. Como pueden visualizar en el siguiente ejemplo en donde se hace dificil leer el código.

Segunda forma "Usar CSS dentro de la etiqueta head con la etiqueta style", esta forma presenta una mayor funcionalidad y mantenibilidad del código css ya que nos mezclamos CSS, dentro de cada etiqueta HTML, ahora lo que se busca es colocar todos los estilos dentro de la etiqueta style, y esta etiqueta que contiene todos los estilos estara dentro de la etiqueta head, donde estan los metadatos, observa el siguiente ejemplo.

Tercera forma "Usar CSS en un nuevo archivo con extensión .css y usando la etiqueta link", la etiqueta link nos permitirá unir HTML con un documento con extensión .css esto es lo más recomendable según "MDN MOZILA FOUNDATION".
